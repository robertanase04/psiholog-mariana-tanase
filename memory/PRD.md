# PRD - Site Prezentare Cabinet Psihologic Mariana Tanase

## Problem Statement
Site de prezentare pentru cabinetul psihologic al Marianei Tanase din București, România. Site doar de prezentare, fără backend complex, cu integrare Calendly pentru programări.

## User Personas
- **Clienți potențiali**: Persoane din România care caută servicii de psihologie clinică, evaluări pentru permis auto, port-armă, angajare
- **Pacienți existenți**: Care doresc să programeze ședințe online

## Core Requirements (Static)
1. Site single-page responsive
2. Secțiuni: Hero, Despre Mine, Servicii, Learning, Prețuri, Programări, Contact
3. Integrare Calendly pentru programări
4. Formular de contact funcțional
5. Design calm și profesional (olive, cărămiziu, bej)
6. Mobile-first responsive

## Technical Stack
- React 19 + Tailwind CSS
- react-calendly pentru integrarea programărilor
- framer-motion pentru animații
- lucide-react pentru iconițe
- sonner pentru toast notifications
